# Smart Nutrition Tracker

## Overview
Smart Nutrition Tracker is a responsive web & mobile application that helps users achieve and monitor nutrition goals through AI-driven personalized meal plans, automated grocery lists, and image-based meal recognition.

## Components
- **Backend:** Python (FastAPI) - Located in `/backend`
- **Mobile:** Flutter - Located in `/mobile`

## Getting Started

### Backend
1.  Navigate to `backend/`
2.  Install dependencies: `pip install -r requirements.txt`
3.  Run server: `uvicorn main:app --reload`

### Mobile
1.  Navigate to `mobile/`
2.  Run `flutter pub get`
3.  Run `flutter run`
